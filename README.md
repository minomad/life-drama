# 과제-03

> 꼭 봐야할 인생 드라마를 추천하고 리뷰를 쓰는 사이트

## 구성

홈 - 드라마 - 글쓰기 - 마이페이지

## 기능

- 로그인 / 회원가입
- 글쓰기 (드라마 추천 글)
- 작성된 드라마 리스트 렌더링
- 선택한 드라마의 리뷰 작성
